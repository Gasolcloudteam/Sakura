centos7网络IP配置教程

枷锁云-星尘 · 2022-5-30 · 次阅读

在Centos7中配置ip

均优先执行以下操作

通过cd命令进入到network-scripts目录

cd /etc/sysconfig/network-scripts


ls 命令查看当前文件夹的网卡文件

然后通过vi指令编辑网卡文件，本文以ifcfg-ens33为例

这里是以默认未配置的原生网卡做演示

温馨提示：vi编辑器按下键盘 “i”键 进入编辑模式，编辑内容完毕后按键盘 “Esc”键 此时按下 “shift+;:”两个键 输入 “wq”保存并退出

IP分配方式(static/dhcp)

1.默认配置启用网卡，使用动态获取ip的方式

记得“:wq”保存并退出哦

动态ip配置修改项：
ONBOOT=YES #启用网卡
BOOTPROTO=DHCP #ip分配方式 DHCP为动态获取 static为静态获取 这里演示动态获取

1.vi指令编辑网卡文件（vi ifcfg-ens33）

2.按照开头说明修改项编辑，启动虚拟网卡

3.通过service network restart命令重启网卡服务

4.通过ip a命令，查看配置好的网卡动态ip地址

5.使用Windows cmd ，ping一下虚拟机动态ip，查看是否配置成功

以上便是动态ip的配置教程，请仔细阅读，不要忽视细节，谢谢配合！

1.使用静态配置ip的方式

记得“:wq”保存并退出哦

静态ip配置修改/新增项：
ONBOOT=YES #启用网卡
BOOTPROTO=static #ip分配方式 DHCP为动态获取 static为静态获取 这里演示静态获取
IPADDR=172.16.12.99 #172.16.12需要查看电脑vmnet8的网关地址
GATEWAY=172.16.12.1 #同上
NETMASK=255.255.255.0 #子网掩码

1.vi命令编辑网卡

2.参考开头的静态ip配置修改/新增项，按照实际情况修改

3.通过service network restart命令重启网卡服务

4.通过ip a命令查看配置好的网卡静态ip

5.打开Windows cmd，ping虚拟机静态ip查看是否配置成功

关于网络配置针对网卡的常用指令

nmcli device status #查看网卡设备状态

nmcli connection show #查看链接的状态

ip a  #查看网络信息

nmcli device connect ens33  #链接上指定网卡(ens33是网卡名称)

nmcli device disconnect ens33 #断开网卡链接
